🚀 About The Project

Realtime Chat Application is a full-stack web application that enables users to communicate instantly using WebSockets.

The application supports:

Real-time messaging
Typing indicators
Emoji support
Image sharing
Dark/Light mode
Responsive chat interface

This project demonstrates frontend-backend integration and real-time communication using Socket.IO.

✨ Features
⚡ Instant real-time messaging
👤 Multiple user support
✍️ Typing status indicator
😊 Emoji picker integration
🖼️ Send and preview images
🌙 Dark & Light theme toggle
📱 Fully responsive UI
💬 Modern chat bubble design
